# STRINGS - JAVA
Strings é um tipo de variável que NÃO é primitivo, que podem armazenar um texto.

## 1) Como declarar uma String:
Como String é uma classe em Java, ela precisa começar com a letra MAIÚSCULA.

Exemplo:
```Java
String nome;
```

## 2) Como iniciar uma String:
Exemplo:
```Java
String nome = "Leandro";
```

Printando na tela:
```Java
String nome = "Vasco";
System.out.println("O maior clube do mundo é o "+nome);
```

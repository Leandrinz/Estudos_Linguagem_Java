# ESTRUTURA CONDICIONAL IF - JAVA
A estrutura condicional if (que significa "se") serve para fazer o seu programa tomar decisões. 

## 1) Como funciona:
O if é uma estrutura que possui um bloco que só será executado caso uma condição retorne true, mas caso for false, o bloco será ignorado

## 2) Sintaxe:
```Java
if (condição){
    // bloco do if
        }
```

## Exemplo:
```Java
int golsVasco = 7;
int golsFlamengo = 1;
if (golsVasco > golsFlamengo){ // True
    System.out.println("Vasco ganhou"); // Saída 
        }
```

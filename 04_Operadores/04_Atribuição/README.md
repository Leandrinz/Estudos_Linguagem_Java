# OPERADORES DE ATRIBUIÇÃO - JAVA
Operadores de atribuição em Java são símbolos usados para definir ou atualizar o valor de uma variável.

A seguir, iremos ver seus tipos, utilidades e propriedades

### 1) Operador de atribuição simples (=):
Atribui um valor a uma variável

Exemplo:
```Java
int titulosVasco = 10000;
```

### 2) Incremento (+=):
Acrescentamos um valor a variável

Exemplo:
```Java
int golsVasco = 3;
golsVasco += 1; // Incrementamos 1 a variável, ficando com 4
```

### 3) Decremento (-=):
Diminuímos um valor da variável

Exemplo:
```Java
int golsFlamengo = 2;
golsFlamengo -= 2; // Diminuímos 2 da variável, ficando com 0 (O Vasco vence)
```

### 4) Atribuição do resultado de divisão (/=):
Atribuímos à variável o valor da divisão por um número x

Exemplo:
```Java
int dinheiro = 1000;
dinheiro /= 2; // A variável agora fica com o resultado de 1000 / 2, ou seja, 500.
```

### 5) Atribuição do resto da divisão (%=):
Atribui à variável o valor do resto da divisão por um número x

Exemplo:
```Java
int dinheiro = 11;
dinheiro %= 2; // A variável agora fica com valor 1, que é o resto da divisão de 11 por 2
```

### 6) Atribuição de multiplicação (*=):
Atribui à variável o valor da multiplicação de seu valor por um número x

Exemplo:
```Java
int numero = 10;
numero *= 2; // Agora a variável fica com valor 20, que é o resultado de 10 * 2
```

# OPERADORES ARITMÉTICOS - JAVA
No Java, existem diversos operadores que usaremos nos nossos estudos e desenvolvimento. Mas agora iremos nos aprofundar e conhecer os operadores **Aritméticos**

## 1) Tipos:

#### 1.1) Soma (+):
Soma dois valores

Exemplo:
```Java
int numero1 = 20;
int numero2 = 25;
int soma = numero1 + numero2; // Valor da variável "Soma" = 45
```

#### 1.2) Subtração (-):
Subtrai dois valores

Exemplo:
```Java
int numero1 = 20;
int numero2 = 10;
int subtração = numero1 - numero2; // Valor da variável "Subtração" = 10
```

#### 1.3) Divisão (/):
Faz a divisão de dois valores

Exemplo:
```Java
int numero1 = 10;
int numero2 = 5;
int divisão = numero1 / numero2; // Valor da variável "Divisão" = 2 (Lembrando que é de divisão inteira)
```

#### 1.4) Multiplicação (*):
Faz a multiplicação de dois valores

Exemplo:
```Java
int numero1 = 5;
int numero2 = 4;
int multiplicação = numero1 * numero2; // Valor da variável "Multiplicação" = 20
```

#### 1.5) Resto de Divisão (%):
Mostra o resto de uma divisão inteira

Exemplo:
```Java
int resultado = 10 % 3; // Retorna 1 (o resto que sobrou)
```

# COMENTÁRIOS - JAVA
Comentários são linhas de código que não afetam em nada no nosso programa. Usados geralmente para uma documentação e organização do código e para comunicação entre desenvolvedores que trabalham no mesmo código fonte

Temos alguns tipos de comentários

## 1) Comentário simples - //:

Exemplo: 

```Java
// Isso é um comentário simples
```

## 2) Comentário de múltiplas linhas - /* */:

Exemplo:

```Java
/*
Isso é um comentário de 
múltiplas linhas        
 */
```

## 3) Java Doc - /** * */:

Exemplo:

```Java
/**
 * Isso é um comentário javadoc
 */
```

Agora cuidado. Evite ao máximo usar comentários. Se você usar comentários com frequência, seu código ficará poluído.
